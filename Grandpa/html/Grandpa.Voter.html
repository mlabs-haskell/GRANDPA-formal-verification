Voter

Some requirements about a type that can represent voters:
  • It must have infinite inhabitants.
  • It must have decidable equality.
This means, we can have any number of voters and we can distinguish between them. For this reason we choose to use naturals.
Additionally we use the newtype pattern as we would work a lot with other types isomorphic to naturals.
We aren't using the NatWrapper module here as it doesn't make sense to make operations over the voters other than equality.
Record Voter : Type := VoterC { to_nat:nat }.

Definition from_nat (n:nat) : Voter
  :=
  VoterC n.
